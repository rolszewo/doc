# vim

## indent / unindent

key   | description
----  |------------
`>>`  | indent current line
`5>>` | indent 5 lines
`3<<` | un-indent 3 lines

Lines are shifted 'shiftwidth' to the right or left.

Can be used with visual mode too ( CTRL-v / SHIFT-v )
